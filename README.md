# vue-koa-chatroom

> 这是一个前端用vue后端用koa的一个聊天室

## 实现功能
* 时时通知
* 改变字体大小，颜色，加粗，下划线，倾斜
* 异步上传图片

## Build Setup

``` bash
# install dependencies
cd server && npm install
cd client && npm install

# start server
cd server && npm start
server start at localhost:3000

# start client
cd cilent && npm run dev
server start at localhost:8080

```
